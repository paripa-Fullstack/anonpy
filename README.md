Phantom Net-Stressor — это высокопроизводительный многопоточный оркестратор для симуляции распределенных атак типа «Отказ в обслуживании» (DoS/DDoS) на уровнях L3, L4 и L7. Инструмент предназначен для специалистов по информационной безопасности, Red Team операторов и сетевых инженеров с целью стресс-тестирования WAF, IPS/IDS систем, rate-limiting политик и общей отказоустойчивости инфраструктуры.

🧬 Архитектура и Ключевые Особенности
1. Мультивекторный Движок (L3 / L4 / L7)
SYN Flood (L4): Использование Raw Sockets (IP_HDRINCL) для ручного крафтинга IP и TCP заголовков. Поддержка рандомизации Source IP (Spoofing) и TCP Sequence Numbers для обхода stateful фаерволов.
UDP Flood (L4): Генерация высокоэнтропийных пейлоадов (os.urandom) для исчерпания полосы пропускания (Bandwidth Exhaustion).
HTTP/HTTPS Flood (L7): Симуляция легитимного L7-трафика с использованием пула сессий requests, рандомизацией URI (Cache-Busting) и подделкой L7-заголовков (X-Forwarded-For, Client-IP).
Slowloris (L7): Исчерпание пула соединений веб-сервера (Connection Pool Exhaustion) путем отправки частичных HTTP-запросов с удержанием Keep-Alive сокетов.
DNS Amplification (L3): Симуляция отраженных атак с использованием открытых резолверов и ANY-запросов.
2. Продвинутый OpSec и Proxy Chaining
Кастомный SOCKS-движок (socks.py): Глубокая интеграция с протоколами SOCKS4/4a/5 и HTTP CONNECT. Поддержка UDP ASSOCIATE для проксирования L4-трафика через SOCKS5.
Tor Network Integration: Нативная маршрутизация через socks5h (с удаленным DNS-резолвингом) для полной анонимизации L7-векторов.
Header Spoofing: Автоматическая инжекция фальшивых X-Real-IP и X-Forwarded-For для тестирования WAF на уязвимости IP-Spoofing.
3. Out-of-Band C&C (Command & Control)
Telegram Bot API: Встроенный модуль для удаленного управления ботнетом/стрессором через защищенный канал. Поддержка динамической смены цели (/target) и экстренной остановки (/stopatt) без необходимости прямого SSH-доступа к серверу-источнику.

⚙️ Установка и Зависимости
Для работы L3/L4 векторов (SYN Flood) требуются root-привилегии (или sudo), так как используется создание Raw Sockets.
# Клонирование репозитория
git clone https://github.com/your-username/phantom-net-stressor.git
cd phantom-net-stressor

# Установка зависимостей
pip install requests pysocks urllib3

# Запуск (Требуется Root для SYN/UDP Raw Sockets)
sudo python3 anon.py

🛡️ Mitigation & Defensive Recommendations
Используя данный инструмент, Blue Team инженеры должны настраивать следующие защиты:
SYN Cookies & TCP Interception: Включить на уровне пограничных роутеров (Cisco/Juliper) или Linux net.ipv4.tcp_syncookies = 1.
L7 Rate Limiting: Настройка limit_req в Nginx или правил Rate-Limiting в Cloudflare/AWS WAF на основе JA3 и поведенческого анализа.
Egress/Ingress Filtering: Реализация BCP 38 (RFC 2827) для блокировки пакетов со поддельными Source IP.
Connection Timeouts: Агрессивные таймауты для неактивных HTTP-сессий (защита от Slowloris).
